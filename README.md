# Deskripsi Singkat Cara Menjalankan File

Jalankan Program di WSL
Pastikan WSL sudah terpasang dan kamu berada di direktori if1210-tubes-2025-k01-n
Jalankan perintah make di WSL untuk compile program:
make

Jalankan program hasil compile di WSL dengan perintah:
./main <namafolder>
misal: ./main ori

Ikuti instruksi pada layar untuk menggunakan fitur program seperti melihat denah rumah sakit, detail ruangan, antrian pasien, dan inventaris obat. Pilih menu & help untuk penjelasan lanjutan.
